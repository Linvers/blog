### 安装git
下载git

### 安装nodejs
npm下载的时候推荐用淘宝npm镜像

```
npm install -g cnpm --registry=https://registry.npm.taobao.org
```

### 安装hexo
```
利用npm命令就可 
npm install -g hexo
```

### 创建hexo文件夹
```
hexo init
```

### 安装依赖包
```
npm install
```

### 在本地查看
```
执行
hexo generate 
hexo server
```

然后再localhost:4000看看
